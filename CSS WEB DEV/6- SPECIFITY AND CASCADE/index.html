SPECIFITY:- CSS specifity determines which style rule applied to an element when there are coflicts. Higher specifity means the style will be used. It is calculated based on a point system involving inline styles.

CASCADE:- The cascade is the algorithm for solving conflicts where multiple CSS rules apply to an HTML element.

In the above heading a single element has given three classes and all three classes have assigned different properties. According to the specifity and cascade rules, the last class will be applied to the element. point to remember is that three selectors are same so last one is applied. If the selectors are same then cascading is calculated. And when the selectors are different then specifity is calculated.

The order of specifity is given as follows:-
Inline Style > ID Selector > Class or Attribute Selector > Element Selector > universal Selector >
