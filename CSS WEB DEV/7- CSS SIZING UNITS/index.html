CSS Size:- CSS has several different units for expressing a length.

Absolute units
  1. cm = centimetre
  2. mm = milimetre
  3. in = inches
  4. pt = points
  5. pc = picas
  6. px= pixel
Generally we use pixel(px) to set dimensions of elements.
