CSS Variable

VARIABLE:- CSS variable or custom properties are entities defined by CSS authors that reperesents specific values to be reused throughout a document. Variables in CSS helps us to assign the same properties to different elements. Any CSS variable written in the root variable can be accessed anywhere in the code.
